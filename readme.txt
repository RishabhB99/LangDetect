Before running the program please install the following prerequisites:
	pip3 install pycld3
	pip3 install langdetect
	pip3 install fasttext

I have provided models langid and fasttext used. Please download these files as well( langid.py and lid.176.fitz )
Use python3 detector.py --help for instructions (Add flags in order they are mentioned)	
I have also added sample datatsets if you want to use them
